## Titanic Survival Prediction Project

This project aims to analyze and predict passenger survival on the Titanic using machine learning techniques.

A logistic regression model was built using features such as passenger class, gender, age, and engineered variables including family size and social titles extracted from names.

The model achieved approximately 82% accuracy, demonstrating strong predictive performance for a simple interpretable model.

Key findings show that gender is the most influential factor, with female passengers having significantly higher survival odds. Additionally, titles such as "Mrs" and "Miss" further improve prediction performance, indicating the importance of social roles and demographics.

This project highlights how feature engineering and proper data preprocessing can significantly improve model performance, even with simple algorithms.

Overall, the study demonstrates the importance of combining domain understanding with statistical modeling to derive meaningful insights from data.
